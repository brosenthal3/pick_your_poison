# Pick Your Poison
This is a project made for the Human Computer Interaction and Information Visualization course (24/25) at Leiden University, as a collaboration between Alice Grama and Ben Rosenthal.

## Project Summary 
Pick Your Poison is a web application designed to improve understanding of mushroom species and their toxicity by taking an interactive and exploratory approach. The app allows users to create custom virtual mushrooms through dynamic visualization and playful exploration. The underlying machine learning classifiers predict whether the user's designs resemble poisonous or non-poisonous species and identify the most similar real-life counterparts. 
Unlike traditional plant identification tools, Pick Your Poison emphasizes exploratory learning over direct identification, addressing the limitations of image-based apps. The app was tested in two rounds of user evaluations that showed high satisfaction scores and effectiveness. The usability specifications were met in both evaluations. Although the project has limitations such as a small number of testers and not fully fledged educational features, we can conclude that Pick Your Poison is a fun and interactive tool for exploring mushroom toxicity and species.

## This repository
This repository contains the front-end implementation of Pick Your Poison, written mostly in Flutter. The backend API can be found in https://github.com/brosenthal3/pyp_api.
